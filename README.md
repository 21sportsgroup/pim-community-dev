TranslationBundle
=================

Deal with i18n datas

Install
=======
To install for dev :

```bash
$ php composer.phar update --dev
```

To use as dependency, use composer and add bundle in AppKernel :

```json
    "require": {
        [...]
        "pim/TranslationBundle": "dev-master"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:akeneo/TranslationBundle.git",
            "branch": "master"
        }
    ]
```
